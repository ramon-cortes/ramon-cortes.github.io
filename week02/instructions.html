How to create a web page in GitHub

You can use GitHub to create a personal web page. Follow the steps to get it done.

Create a GitHub account

Enter github.com and on the top right click on "Sign up"
Follow the steps:

  • Email
  • Password
  • Username

You will receive a welcome email with an email verification code to continue.

Create a Repositorie

Once you log into your GitHub there are some ways to create your Repositorie. One way is clicking on the top right corner on your user "icon" and then click on "Your repositories".

On the right sight click on "New"

Just fill the Field "Repositorie Name" with your_username.github.io
Make sure "Public" is selected and click "Create repositorie".

Upload files

Upload your files by clicking on "upload an existing file". After you have dragged and dropped your files (using "index.html" as your main page) click at the bottom of the page on "Commit changes".

Test your web page

In a new tab access the page your_username.github.io it may take a few minutes for changes to be reflected.
